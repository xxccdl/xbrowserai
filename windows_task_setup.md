# Windows定时任务设置指南

## 设置每分钟查看新闻的定时任务

### 步骤1：准备脚本文件
确保 `news_checker.bat` 文件在您知道的位置，例如：`C:\Users\您的用户名\Desktop\news_checker.bat`

### 步骤2：打开任务计划程序
1. 按 `Win + R` 打开运行对话框
2. 输入 `taskschd.msc` 并按回车

### 步骤3：创建基本任务
1. 在右侧操作面板点击"创建基本任务"
2. 输入名称：`每分钟查看新闻`
3. 描述：`自动每分钟打开新闻网站`
4. 点击"下一步"

### 步骤4：设置触发器
1. 选择"每天"
2. 点击"下一步"
3. 设置开始时间（如当前时间）
4. 勾选"重复任务间隔"，设置为1分钟
5. 持续时间：选择"无限期"
6. 点击"下一步"

### 步骤5：设置操作
1. 选择"启动程序"
2. 点击"下一步"
3. 程序或脚本：浏览并选择 `news_checker.bat` 文件
4. 起始于：填写脚本所在目录（如 `C:\Users\您的用户名\Desktop`）
5. 点击"下一步"

### 步骤6：完成设置
1. 查看摘要信息
2. 点击"完成"

### 步骤7：测试任务
1. 在任务计划程序库中找到刚创建的任务
2. 右键点击任务，选择"运行"
3. 检查是否成功打开新闻网站

## 高级设置（可选）

### 修改触发器设置：
1. 双击任务打开属性
2. 切换到"触发器"选项卡
3. 双击现有触发器进行编辑
4. 可以设置：
   - 开始日期和时间
   - 重复间隔（1分钟）
   - 持续时间（如8小时，避免夜间打扰）

### 条件设置：
1. 切换到"条件"选项卡
2. 可以设置：
   - 只在计算机使用交流电源时运行（笔记本）
   - 只有在计算机空闲时启动
   - 唤醒计算机运行此任务

### 设置设置：
1. 切换到"设置"选项卡
2. 可以设置：
   - 允许按需运行任务
   - 如果任务运行时间超过以下时间，停止任务：设置为2分钟
   - 如果任务没有按计划启动，立即启动任务

## 注意事项

1. **浏览器设置**：确保默认浏览器已设置
2. **网络连接**：需要网络连接才能打开新闻网站
3. **系统权限**：可能需要管理员权限
4. **资源占用**：每分钟打开网页可能会占用较多资源，建议在工作时间使用
5. **停止任务**：如需停止，在任务计划程序中右键任务选择"禁用"

## 自定义新闻网站

要修改查看的新闻网站，编辑 `news_checker.bat` 文件，将 `https://news.baidu.com` 替换为您喜欢的新闻网站URL。

推荐新闻网站：
- 百度新闻：https://news.baidu.com
- 网易新闻：https://www.163.com
- 新浪新闻：https://www.sina.com.cn
- 腾讯新闻：https://news.qq.com
